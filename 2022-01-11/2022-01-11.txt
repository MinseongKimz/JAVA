switch 
○다중 선택문이라 하며, switch 문 다음위 수식 값에 따라
  실행 순서를 여러 방향으로 분기할 때 사용하는 문장이다/
  
○형식 및 구조
	switch(수식)
	{
		case 상수1 : 문장1; [break;]
		case 상수2 : 문장2; [break;]
					...
		[default : 문장 n+1;[break;]] // 없다고 잘못된건 아님
	}

 『switch』 문의 수식과 『case』의 상수는 
 byte, short, int, long형이여야 한다.

 case문 뒤에 [break;]가 없는 형태인 경우
 다음 case문의 문장을 계속해서 수행하게 된다; >> 기본 모델
 [break;]가 존재할 경우
 해당 지점에서 수행을 멈춘 후 switch 문을 빠져나간다 >> 일반모델



for, while 그리고 do~while 반복문

1.while 문
 while (조건식) //조건식의 결과는 true or false, if문과 같음
 {
  // 처리 구문;
 }

조건이 아니면 안하는건 if 문과 같다. 나감
하지만 while문 과 if문의 차이는 while문은 조건식이 true면
처리 구문을 실행하고 if문처럼 빠져나가지 않고 
다시 조건식을 보러 가고 처리구문을 실행한다.
이걸 조건식이 false가 될 때 까지 반복한다.
조건에 따라 한번도 실행되지 않을 경우도 있다.

자주 쓰는 경우 : 돌려놓고 특정 상황에 끌떄?

2. do~while문
 do
 {
  // 처리 구문
 }
 while(조건식)

일단 한번은 무조건 실행! 그다음에 조건반복

3. for 문
 for (수식1; 수식2; 수식3)
 {
  //처리 구문
 }
수식1 : 제일먼저 수행 ( 변수 선언)

수식2 : true false > 트루면 처리구문 > 처리 구문 끝나면 수식3
수식3 : 증감연산 > 수식2 
수식2 3 에서 돌다가 2에서 false 나오면 for문 종료

무한루프 만드는법
while문 은 조건식에 true
for문은 수식123 다 비우면 된다
